Md. Rezve Hasan

Lead Engineer @ Brain Station 23

I started loving programming in 2010 after seeing my elder brother's calculator application in the terminal. I began reading The C Programming Language by Dennis Ritchie and wrote code on paper, desperate to understand the magic.

Later, I moved into web development—HTML, CSS, PHP, and SQL—developing a couple of web applications. To deepen my understanding of OOP, I started learning Java, where I developed two games and a few desktop applications.

After realizing all the problems I could solve by writing code, I decided to pursue Computer Science Engineering. Besides studying core subjects, I explored various development areas, including a few Android apps, games in Unity and c++, Node.js and MongoDB.

I started as a Junior Software Engineer in 2018 and now continue as a Lead Engineer.

I can remember, I was always in the mindset of 'If it takes code, I can program it.' But now, I find myself very close to the business side of software development. Everything I do is about the value I bring to the end user. Every line of code I write is aimed at solving a problem, and every tradeoff I consider is based on its impact on the users and the business.

I moved from writing code for the sake of it to writing code that matters—code that brings real value to users and businesses.
